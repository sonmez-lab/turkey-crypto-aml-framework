# Contributing to This Project

Thank you for your interest in contributing! This project aims to make AML/CFT compliance tools accessible to organizations of all sizes.

## How to Contribute

### Reporting Issues
- Use GitHub Issues to report bugs or suggest features
- Include as much detail as possible
- For security vulnerabilities, please email privately

### Pull Requests
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### Code Style
- Follow existing code conventions
- Add tests for new functionality
- Update documentation as needed

### Areas We Need Help

## Code of Conduct

Be respectful and constructive. We're building tools to fight financial crime together.
